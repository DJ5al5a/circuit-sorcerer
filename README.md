# Circuit Sorcerer Portfolio

Personal portfolio website showcasing homelab infrastructure, security projects, and technical expertise.

**Live Site:** https://circuitsorcerer.us.kg

## Tech Stack

- **Framework:** Next.js 16 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Features

### Frontend
- Dark industrial tech theme
- Dashboard card-based layout
- Project showcase with filtering
- Skills and certifications display
- Responsive design
- Static export optimized

### Backend
- Contact form with email notifications
- Media request system
- Emby user registration with admin approval
- n8n workflow automation
- PostgreSQL database integration
- Discord & Telegram notifications

**Backend Workflows:** https://github.com/DJ5al5a/n8n-workflows

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
